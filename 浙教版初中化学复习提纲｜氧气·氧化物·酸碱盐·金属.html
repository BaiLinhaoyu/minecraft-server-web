浙江中考 氧化物 酸碱盐 金属 高中拓展
▶ 氧气 O₂ 非金属单质 中考必考
▶ 物理性质（详细）
• 颜色状态：通常状况下为无色、无味的气体
• 密度：比空气略大（相对分子质量32 > 空气29），因此可用向上排空气法收集
• 溶解性：不易溶于水（1L水约溶30mL O₂），因此可用排水法收集，且比用排空气法收集的更纯净
• 三态变化：降温加压可变为淡蓝色液体（液氧，-183°C）或淡蓝色雪花状固体（固氧，-218°C）
• 工业储存：通常加压贮存在蓝色钢瓶中（液氧）
▶ 化学性质（比较活泼，支持燃烧，供给呼吸） 高频实验
▶ 与非金属反应（燃烧现象对比）
1. 木炭（碳）燃烧：
C + O₂ = CO₂（点燃，充分燃烧）
• 空气中：持续红热，放出热量，生成能使澄清石灰水变浑浊的气体
• 氧气中：剧烈燃烧，发出白光，放出大量的热，生成能使澄清石灰水变浑浊的气体
• 注意：若氧气不足，不充分燃烧：2C + O₂ = 2CO（点燃）
2. 硫燃烧（现象对比明显，中考高频）：
S + O₂ = SO₂（点燃）
• 空气中：发出微弱的淡蓝色火焰，放出热量，生成有刺激性气味的气体
• 氧气中：发出明亮的蓝紫色火焰，放出大量的热，生成有刺激性气味的气体
• 注意：集气瓶底要放少量水或氢氧化钠溶液，吸收SO₂防止污染空气（SO₂有毒，形成酸雨）
3. 红磷（白磷）燃烧（测定空气中氧气含量实验）：
4P + 5O₂ = 2P₂O₅（点燃）
• 现象：产生大量白烟（五氧化二磷固体小颗粒），放出大量的热，生成白色固体
• 注意：是"白烟"（固体）不是"白雾"（液体）；P₂O₅有毒，实验需在密闭容器中进行
4. 氢气燃烧（清洁能源）：
2H₂ + O₂ = 2H₂O（点燃）
• 现象：产生淡蓝色火焰，放出大量的热，烧杯内壁有水珠生成（火焰上方罩干冷烧杯）
• 注意：点燃前必须验纯（爆鸣声），否则可能爆炸
▶ 与金属反应（剧烈氧化，中考重点实验）
1. 铁丝燃烧（中考必考实验，现象剧烈）：
3Fe + 2O₂ = Fe₃O₄（点燃）
• 现象：剧烈燃烧，火星四射，放出大量的热，生成黑色固体（四氧化三铁）
• 操作要点：①铁丝绕成螺旋状（增大受热面积，聚集热量）；②末端系一根火柴（引燃）；③待火柴快燃尽时再伸入集气瓶（防止火柴燃烧消耗过多氧气）；④瓶底放少量水或细沙（防止高温熔融物溅落炸裂瓶底）
• 注意：铁丝在空气中不能燃烧，只能烧至红热；铁丝表面若有铁锈需用砂纸打磨除去
2. 镁条燃烧（耀眼白光）：
2Mg + O₂ = 2MgO（点燃）
• 现象：发出耀眼的白光，放出大量的热，生成白色固体（氧化镁），冒白烟
• 应用：制照明弹、闪光粉、焰火（利用其耀眼白光）
• 注意：镁在空气中即可燃烧，氧气中更剧烈；不要直视强光以免损伤眼睛
3. 铝燃烧（高中拓展）：
4Al + 3O₂ = 2Al₂O₃（点燃）
• 现象：剧烈燃烧，发出耀眼的白光，放出大量的热，生成白色固体
• 注意：铝在空气中表面有氧化膜阻碍，需打磨或高温点燃；氧气中燃烧更剧烈
4. 铜与氧气（加热，缓慢氧化）：
2Cu + O₂ = 2CuO（加热）
• 现象：紫红色固体表面逐渐变为黑色（生成氧化铜）
▶ 与化合物反应（一氧化碳、甲烷等）
1. 一氧化碳燃烧（蓝色火焰）：
2CO + O₂ = 2CO₂（点燃）
• 现象：产生蓝色火焰，放出大量的热，生成能使澄清石灰水变浑浊的气体
• 注意：CO有毒，点燃前需验纯，尾气需处理
2. 甲烷（天然气）燃烧：
CH₄ + 2O₂ = CO₂ + 2H₂O（点燃）
• 现象：产生明亮的蓝色火焰，放出大量的热，烧杯内壁有水珠生成，倒入澄清石灰水变浑浊
• 注意：点燃前必须验纯（防止爆炸）
氧化反应总结：物质与氧发生的反应叫做氧化反应。氧气在氧化反应中提供氧，具有氧化性（助燃性），是一种常用的氧化剂。

▶ 实验室制法（化学变化，中考必考三大方法）
方法一：加热高锰酸钾（KMnO₄，传统方法，装置较复杂）
2KMnO₄ = K₂MnO₄ + MnO₂ + O₂↑（加热，Δ）
• 药品：暗紫色（紫黑色）固体高锰酸钾
• 装置：固体加热型（试管口略向下倾斜，防止冷凝水倒流炸裂试管）
• 操作要点：①试管口塞一团棉花（防止KMnO₄粉末进入导管堵塞或污染水槽）；②铁夹夹在距试管口约1/3处；③先预热后集中加热；④实验结束时先将导管移出水面再熄灭酒精灯（防止水倒吸炸裂试管）
• 收集：排水法（较纯净）或向上排空气法
方法二：分解过氧化氢（H₂O₂，双氧水，常温进行，装置简单，推荐方法）
2H₂O₂ = 2H₂O + O₂↑（MnO₂催化）
• 药品：过氧化氢溶液（双氧水，无色液体，通常6%浓度）和二氧化锰（黑色固体催化剂）
• 装置：固液不加热型（锥形瓶/试管 + 长颈漏斗/分液漏斗）
• 催化剂（MnO₂）："一变两不变"——改变化学反应速率（加快），本身的质量和化学性质在反应前后不变
• 操作：可通过分液漏斗控制滴加速度，从而控制反应速率；也可用注射器代替
• 优点：节能（不需加热）、环保（产物只有水和氧气）、操作简便、可控性强
方法三：加热氯酸钾和二氧化锰混合物（KClO₃，较少用，了解）
2KClO₃ = 2KCl + 3O₂↑（MnO₂催化，加热）
• 注意：MnO₂是催化剂，加快反应速率但质量和化学性质不变；反应需加热，装置同KMnO₄法但试管口不需塞棉花
▶ 工业制法（物理变化，分离液态空气法）
• 原理：利用液态氮（沸点-196°C）和液态氧（沸点-183°C）的沸点不同，通过蒸发液态空气分离（氮气先蒸发，剩余主要是液氧）
• 过程：空气净化 → 加压降温液化 → 蒸发分离（氮气沸点低先蒸发）→ 获得液氧（贮存在蓝色钢瓶中）
• 变化类型：物理变化（无新物质生成，只是状态变化和分离）
• 储存：加压贮存在蓝色钢瓶中，便于运输和使用
▶ 检验、验满与收集方法（中考操作必考）
1. 检验方法（证明是氧气）：
将带火星的木条伸入集气瓶内，若木条复燃，则证明是氧气
• 原理：氧气支持燃烧，浓度越高燃烧越旺
2. 验满方法（证明已收集满）：
• 向上排空气法：将带火星的木条放在集气瓶口，若木条复燃，则证明已收集满
• 排水法：当集气瓶口有大气泡向外冒出时，证明已收集满（或瓶内水已排尽）
3. 收集方法选择：
• 排水法：因为O₂不易溶于水，且不与水反应；此法收集的氧气较纯净（但含有水蒸气）
• 向上排空气法：因为O₂密度比空气大；此法收集的氧气较干燥（但可能混有空气，纯度稍低）
• 最佳组合：如需干燥氧气，先排水法收集再干燥；或向上排空气法+干燥剂
▶ 用途与氧化反应类型
1. 供给呼吸（生物必需）：
• 医疗急救（氧气瓶、呼吸机）
• 潜水（氧气瓶）
• 登山（高原缺氧）、航空（飞行员）、航天（宇宙飞船）
• 动植物呼吸作用：C₆H₁₂O₆ + 6O₂ = 6CO₂ + 6H₂O + 能量（酶催化）
2. 支持燃烧（助燃性，非可燃性）：
• 炼钢（提高炉温，使生铁中碳充分燃烧降低含碳量）
• 气焊、气割（乙炔在氧气中燃烧产生高温火焰，3000°C以上）
• 液氧炸药（采矿）
• 火箭推进剂（液氢+液氧）
氧化反应分类（从剧烈程度）：
• 剧烈氧化（燃烧）：发光放热明显，反应剧烈，如物质在氧气中燃烧、爆炸
• 缓慢氧化：进行得很慢，甚至不易察觉，不发光但放热，如铁生锈、食物腐败、动植物的呼吸、酒和醋的酿造、农家肥的腐熟
• 共同点：都是氧化反应，都放出热量；不同点：剧烈程度不同
• 自燃：由缓慢氧化引起的自发燃烧（如白磷自燃，热量积累达到着火点）
▶ 催化剂与催化作用（中考概念辨析）
定义：在化学反应中能改变其他物质的化学反应速率（加快或减慢），而本身的质量和化学性质在反应前后都没有发生变化的物质叫做催化剂（又叫触媒）。催化剂所起的作用叫催化作用。
关键词理解（"一变两不变"）：
• 改变速率：可以加快也可以减慢（如MnO₂加快H₂O₂分解，橡胶中的防老剂减慢氧化）
• 质量不变：反应前后质量相等（物理质量不变，但可能参与中间过程）
• 化学性质不变：反应前后化学性质相同（物理性质可能改变，如颗粒变细小）
• 针对性：催化剂只对特定反应起作用（MnO₂对H₂O₂有效，对其他反应不一定）
• 不参与产物：催化剂不是反应物，也不是生成物，写在箭头上方
• 常见催化剂：MnO₂（过氧化氢、氯酸钾分解）、Fe₂O₃（某些反应）、酶（生物催化剂）
▶ 氧化物 Oxides 二元化合物 中考必考
▶ 非金属氧化物 酸性/不成盐
▶ 一氧化碳 CO 剧毒 中考重点
▶ 物理性质（详细）
• 颜色状态：无色、无味、无臭的气体
• 密度：比空气略小（相对分子质量为28，空气平均为29），可用排水法收集
• 溶解性：难溶于水（1体积水溶解0.02体积CO），这是煤气中毒难以察觉的原因
• 熔沸点：熔点-205°C，沸点-191.5°C
▶ 化学性质（浙江中考三大考点）
1. 可燃性（淡蓝色火焰，放热反应）
2CO + O₂ = 2CO₂（点燃）
• 现象：产生淡蓝色火焰，放出大量的热，生成能使澄清石灰水变浑浊的气体
• 注意：火焰颜色较淡，需在光线较暗处观察；点燃前必须验纯，防止爆炸
• 毒性原理：CO与血红蛋白结合能力比O₂强200-300倍，造成组织缺氧
2. 还原性（冶炼金属，初中重点实验）
CO + CuO = Cu + CO₂（加热）
• 现象：黑色粉末逐渐变为光亮的红色固体，生成的气体能使澄清石灰水变浑浊
• 实验要点：①先通CO后加热（排尽空气，防止爆炸）；②先停热后停CO（防止生成的Cu被氧化）；③尾气处理（点燃或用气球收集，因CO有毒）
3CO + Fe₂O₃ = 2Fe + 3CO₂（高温）
• 工业应用：高炉炼铁原理，CO作为还原剂将铁从铁矿石中还原出来
3. 毒性：煤气中毒的元凶，冬天燃煤取暖需通风
▶ 制备与用途
工业制备（水煤气反应）：
C + H₂O(g) = CO + H₂（高温）
• 说明：g表示气态（gas），即水蒸气；此反应在高温下进行，碳作为还原剂将水还原
• 产物：CO和H₂的混合物称为"水煤气"，是重要的气体燃料和化工原料
实验室制法：
HCOOH = CO↑ + H₂O（浓H₂SO₄，Δ）
• 反应条件：必须标注"浓H₂SO₄"作脱水剂，"Δ"表示加热
• 原理：浓硫酸脱去甲酸（HCOOH）中的H₂O，生成CO
• 收集方法：排水法（因CO难溶于水）或向下排空气法（密度略小于空气）
• 用途：①气体燃料（水煤气）；②冶炼金属的还原剂（铁、铜等）；③化工合成原料（制甲醇、光气等）
▶ 二氧化碳 CO₂ 酸性氧化物 浙江高频
▶ 物理性质（详细）
• 颜色状态：无色无味的气体
• 密度：比空气大（相对分子质量44 > 空气29），因此可用向上排空气法收集
• 溶解性：能溶于水（1体积水溶解1体积CO₂），且与水反应生成碳酸，因此一般不用排水法收集
• 三态变化：加压降温可变为液态；固态CO₂称为"干冰"，升华时吸收大量的热，使周围温度降低，用于人工降雨、舞台烟雾效果、食品冷藏运输
▶ 化学性质（浙江中考必考）
1. 酸性氧化物通性（不能使干燥石蕊变色）
CO₂ + H₂O = H₂CO₃（碳酸，使石蕊试液变红）
• 注意：CO₂本身不能使石蕊变色，是生成的H₂CO₃使石蕊变红；加热后红色褪去（H₂CO₃分解）
CO₂ + 2NaOH = Na₂CO₃ + H₂O（吸收CO₂）
• 应用：用NaOH溶液吸收尾气中的CO₂
CO₂ + Ca(OH)₂ = CaCO₃↓ + H₂O（澄清石灰水变浑浊，检验CO₂）
• 现象：澄清石灰水变浑浊（生成白色沉淀），这是检验CO₂的特征反应
• 过量：继续通CO₂，沉淀溶解：CaCO₃ + H₂O + CO₂ = Ca(HCO₃)₂
2. 氧化性（弱，高温下表现）：
CO₂ + C = 2CO（高温，吸热反应）
• 现象：黑色固体逐渐减少，生成无色气体
CO₂ + 2Mg = 2MgO + C（点燃）
• 特例：镁能在CO₂中燃烧，说明CO₂不支持燃烧是相对的（活泼金属例外）
浙江中考易错点：CO₂能溶于水且与水反应，因此实验室制取CO₂不能用排水法收集，必须用向上排空气法。若用排水法需用饱和碳酸氢钠溶液或快速收集以减少损失。
▶ 制备与检验（浙江实验题高频）
实验室制法（必会）：
CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
• 药品：石灰石（或大理石，主要成分CaCO₃）和稀盐酸
• 装置：固液不加热型（启普发生器或简易装置：锥形瓶+长颈漏斗/分液漏斗）
• 不能用硫酸：CaCO₃ + H₂SO₄ = CaSO₄ + H₂O + CO₂↑，但生成的CaSO₄微溶，覆盖在石灰石表面，阻止反应继续
• 不能用浓盐酸：浓盐酸有挥发性，使CO₂中混有HCl气体，导致气体不纯
• 收集：向上排空气法（密度大于空气，且能溶于水）
• 验满：将燃着的木条放在集气瓶口，若木条熄灭，则已集满
• 检验：将气体通入澄清石灰水，若石灰水变浑浊，则为CO₂
• 净化：除HCl用饱和NaHCO₃溶液，干燥用浓H₂SO₄
工业制法（高温煅烧石灰石）：
CaCO₃ = CaO + CO₂↑（高温）
• 此反应也是工业上制生石灰（CaO）和CO₂的方法
▶ 碳循环与温室效应（浙江科学与社会联系）
• 自然界碳循环：植物光合作用吸收CO₂释放O₂（6CO₂ + 6H₂O = C₆H₁₂O₆ + 6O₂，光照、叶绿体），生物呼吸和燃料燃烧释放CO₂
• 温室效应：CO₂、CH₄、N₂O等温室气体导致全球变暖
• 低碳生活措施：①开发新能源（太阳能、风能、氢能）；②植树造林，增加绿化面积；③节能减排，减少化石燃料使用；④乘坐公共交通，减少私家车出行
▶ 其他非金属氧化物 高中拓展
• 二氧化硫 SO₂：无色有刺激性气味的气体，酸性氧化物，具有漂白性（与有色物质化合生成无色物质，可逆，加热恢复原色），是形成酸雨的主要物质之一（煤燃烧产生）
• 三氧化硫 SO₃：标准状况下为固体，与水剧烈反应生成硫酸：SO₃ + H₂O = H₂SO₄
• 氮氧化物 NO/NO₂：NO无色，NO₂红棕色，都是有毒气体，造成光化学烟雾和硝酸型酸雨
• 二氧化硅 SiO₂：原子晶体，硬度大，熔点高，与氢氟酸反应（特性）：SiO₂ + 4HF = SiF₄↑ + 2H₂O，是光导纤维和石英玻璃的主要原料

▶ 氧化钙 CaO（生石灰） 浙江必考
物理性质：白色块状固体，常用作干燥剂（海苔、饼干包装中的小袋子）
化学性质1：与水反应（放热反应，初中重点）
CaO + H₂O = Ca(OH)₂
• 现象：块状固体逐渐变为粉末，放出大量的热（触摸试管壁感到发烫），溶液温度显著升高
• 产物：生成Ca(OH)₂（氢氧化钙，俗称熟石灰、消石灰），微溶于水，水溶液呈碱性
• 用途：食品干燥剂（吸收水分），但需注意：①遇水放热，不可浸水；②不可食用；③包装破损后失效
化学性质2：与酸反应（碱性氧化物通性）
CaO + 2HCl = CaCl₂ + H₂O
• 现象：白色固体逐渐溶解，溶液温度升高（因为CaO与酸反应实质是先与水反应生成Ca(OH)₂，再与酸中和，两步都放热）
浙江中考高频实验：干燥剂变质探究（重点）
• 变质过程：CaO吸收空气中的H₂O生成Ca(OH)₂，Ca(OH)₂再吸收CO₂生成CaCO₃：CaO → Ca(OH)₂ → CaCO₃
• 检验方法：
① 检验是否含CaO：取样加水，若放热（温度计示数升高/触摸发烫），则含CaO；若不放热，则已完全变质
② 检验是否含Ca(OH)₂：取样溶于水（或加水后过滤），取上层清液滴加酚酞试液，若变红，则含Ca(OH)₂
③ 检验是否含CaCO₃：取样加稀盐酸，若有气泡产生（CO₂），则含CaCO₃
• 注意：若有CaO存在，步骤①加水后也会生成Ca(OH)₂，会干扰步骤②中原本Ca(OH)₂的检验，因此需设计实验排除干扰或分别取样
▶ 氧化铁 Fe₂O₃（铁锈主要成分） 浙江必考
物理性质：红棕色（或红色、红褐色）固体，无磁性，铁锈的主要成分（铁锈是Fe₂O₃·nH₂O，含结晶水）
化学性质1：与酸反应（除锈原理，中考高频）
Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O
• 现象：红棕色固体逐渐减少（或逐渐消失），溶液由无色变为黄色（Fe³⁺的颜色）
• 注意：生成的FeCl₃溶液是黄色的，不是浅绿色（浅绿色是Fe²⁺，如FeCl₂）
• 工业应用：用稀盐酸或稀硫酸除去钢铁表面的铁锈，但时间不宜过长，否则内部的铁会继续与酸反应
化学性质2：被还原（高炉炼铁原理，初中重点）
3CO + Fe₂O₃ = 2Fe + 3CO₂（高温）
• 现象：红棕色粉末逐渐变为黑色固体（铁粉），生成的气体能使澄清石灰水变浑浊
• 其他还原剂：也可用H₂或C还原：
Fe₂O₃ + 3H₂ = 2Fe + 3H₂O（加热，试管口向下防炸裂）
2Fe₂O₃ + 3C = 4Fe + 3CO₂↑（高温）
铁生锈条件探究（浙江中考实验探究题高频考点）
• 实质：铁与氧气（O₂）和水（H₂O）同时接触发生的复杂化学反应，是缓慢氧化过程
• 控制变量法实验设计：
① 干燥空气：铁钉放在干燥试管中（无水），不生锈 → 证明水是必要条件
② 煮沸蒸馏水：除去水中溶解的O₂，铁钉浸入后油封（无氧），不生锈 → 证明氧气是必要条件
③ 食盐水：铁钉浸入食盐水中，生锈速度加快 → 证明电解质（NaCl）能加速生锈
• 防锈方法原理：隔绝O₂和H₂O中的至少一种：①涂油（机械零件）；②刷漆（门窗、栏杆）；③镀金属（镀锌、镀铬）；④烤蓝（锯条表面生成致密氧化膜）；⑤制成不锈钢（加入Cr、Ni等改变内部结构）
• 注意：铁锈（Fe₂O₃·nH₂O）结构疏松多孔，不能阻止内部的铁继续与O₂、H₂O接触，因此铁锈会继续向内锈蚀，需及时除去
▶ 氧化铜 CuO 浙江必考
物理性质：黑色粉末状固体（铜丝加热表面生成的黑色物质）
化学性质1：与酸反应
CuO + H₂SO₄ = CuSO₄ + H₂O
• 现象：黑色固体逐渐减少（或逐渐消失），溶液由无色变为蓝色（生成Cu²⁺，硫酸铜溶液呈蓝色）
• 同理：CuO + 2HCl = CuCl₂ + H₂O，溶液变为蓝绿色（氯化铜溶液颜色）
化学性质2：被还原（初中重点实验，H₂、CO、C还原CuO）
CuO + H₂ = Cu + H₂O（加热）
• 现象：黑色固体逐渐变为光亮的红色固体（铜），试管口有水珠生成（H₂O）
• 操作：先通H₂后加热（排空气防爆炸），先停热后停H₂（防生成的Cu被氧化），试管口略向下倾斜（防冷凝水倒流炸裂试管）
CuO + CO = Cu + CO₂（加热）
• 现象：黑色固体逐渐变为光亮的红色固体，生成的气体能使澄清石灰水变浑浊（CO₂）
• 操作：尾气处理（点燃CO，防止污染）
2CuO + C = 2Cu + CO₂↑（高温）
• 现象：黑色固体逐渐变为红色，生成能使澄清石灰水变浑浊的气体
制备方法：铜在空气中加热氧化
2Cu + O₂ = 2CuO（加热）
• 现象：紫红色光亮的铜丝表面逐渐变为黑色
• 铜锈：铜在潮湿空气中与O₂、H₂O、CO₂共同作用生成碱式碳酸铜（铜绿）：2Cu + O₂ + H₂O + CO₂ = Cu₂(OH)₂CO₃（绿色）
▶ 其他碱性氧化物（初中了解）
• 氧化镁 MgO：白色固体，熔点很高（约2800°C），常用作耐火材料（耐火砖、坩埚），与酸反应：MgO + 2HCl = MgCl₂ + H₂O
• 氧化钠 Na₂O/氧化钾 K₂O：白色固体，与水反应比CaO更剧烈，生成NaOH/KOH（强碱），反应放出大量热
▶ 两性氧化物 高中重点
氧化铝 Al₂O₃（刚玉、蓝宝石、红宝石主要成分）：初中教材可能提及其作为铝表面保护膜耐酸碱的性质，高中系统学习其两性
• 与酸反应：Al₂O₃ + 6HCl = 2AlCl₃ + 3H₂O
• 与碱反应：Al₂O₃ + 2NaOH = 2NaAlO₂ + H₂O（或写作Na[Al(OH)₄]，四羟基合铝酸钠）
• 特性：高熔点（约2050°C），硬度大（刚玉），是电解法冶炼铝的原料（溶于冰晶石Na₃AlF₆降低熔点）
氧化锌 ZnO：高中两性氧化物例子，与Al₂O₃类似，既能与酸反应又能与碱反应
▶ 过氧化物与特殊氧化物 高中拓展
• 过氧化钠 Na₂O₂：淡黄色固体，强氧化性，供氧剂（用于呼吸面具、潜水艇、宇宙飞船）
2Na₂O₂ + 2H₂O = 4NaOH + O₂↑
2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂
• 四氧化三铁 Fe₃O₄（磁性氧化铁）：黑色固体，有磁性，可看作FeO·Fe₂O₃（含+2价和+3价铁），是铁在纯氧中燃烧或高温下与水蒸气反应的产物
3Fe + 2O₂ = Fe₃O₄（点燃，火星四射）
3Fe + 4H₂O(g) = Fe₃O₄ + 4H₂（高温）
▶ 酸 Acids 电离出的阳离子有且仅有H⁺ 中考重点
▶ 酸的定义与电离（详细）
酸的定义：电离时产生的阳离子有且仅有氢离子（H⁺）的化合物
• 关键词"有且仅有"：如NaHSO₄（硫酸氢钠）电离时产生Na⁺、H⁺、SO₄²⁻，阳离子除了H⁺还有Na⁺，因此不是酸，而是酸式盐
• 电离方程式示例：
HCl = H⁺ + Cl⁻
H₂SO₄ = 2H⁺ + SO₄²⁻
HNO₃ = H⁺ + NO₃⁻
CH₃COOH ⇌ H⁺ + CH₃COO⁻（可逆，部分电离，弱酸）
▶ 初中三大强酸（必须掌握）
▶ 盐酸 HCl（氢氯酸）
• 物理性质：氯化氢（HCl）气体的水溶液，无色液体，有刺激性气味；浓盐酸（质量分数约37%）具有强挥发性，打开瓶盖瓶口有白雾（HCl气体与空气中水蒸气结合形成的盐酸小液滴）
• 化学性质：具有酸的通性，与金属氧化物反应：Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O
• 用途：金属除锈、制造药物（如盐酸麻黄碱）、胃液中含稀盐酸（约0.5%）帮助消化
▶ 硫酸 H₂SO₄ 浙江高频
物理性质：无色、粘稠、油状液体，不易挥发（高沸点），密度比水大（浓硫酸密度约1.84g/cm³），溶于水放出大量的热（稀释时必须酸入水！）
浓硫酸特性（初中三大特性，中考高频）：
1. 吸水性（物理性质）：能吸收空气中的水分或其他物质中的游离水，常用作干燥剂，可干燥中性气体（O₂、H₂、N₂、CO、CH₄）和酸性气体（CO₂、HCl、SO₂），不能干燥碱性气体（NH₃），因为会发生反应：H₂SO₄ + 2NH₃ = (NH₄)₂SO₄
2. 脱水性（化学性质）：能将有机物中的氢、氧元素按水的组成比（2:1）脱去，使物质碳化变黑。如将浓硫酸滴在蔗糖上：蔗糖逐渐变黑、膨胀，形成疏松多孔的"黑面包"（同时发生氧化还原反应产生刺激性气味气体SO₂）
3. 强氧化性：与金属反应不生成H₂（高中系统学习），常温下使铁、铝钝化（表面形成致密氧化膜，阻止反应继续，因此可用铁/铝槽车运输浓硫酸）
4. 稀释操作（必考）：一定要把浓硫酸沿器壁慢慢注入水中，并不断用玻璃棒搅拌，使热量及时散失。绝对不能把水倒入浓硫酸中，否则水密度小浮在表面，浓硫酸溶于水放热使水沸腾，导致液滴飞溅伤人
▶ 硝酸 HNO₃ 高中重点
• 物理性质：无色液体，有刺激性气味，易挥发，见光或受热易分解：4HNO₃ = 4NO₂↑ + O₂↑ + 2H₂O（光照/加热），因此保存在棕色试剂瓶中并放在阴凉处
• 化学性质：强氧化性酸，与金属反应不生成H₂（无论浓稀，无论金属活泼性），初中了解即可
▶ 初中常见弱酸
▶ 碳酸 H₂CO₃
• 不稳定性：易分解：H₂CO₃ = H₂O + CO₂↑，因此碳酸饮料打开后冒泡，加热后酸性消失
• 弱酸性：使石蕊试液变红，但加热后又变回紫色（CO₂逸出）
▶ 醋酸 CH₃COOH（乙酸）
• 物理性质：无色液体，有强烈刺激性气味（醋味），是食醋的主要成分（约3%-5%）
• 用途：除去水垢（主要成分CaCO₃和Mg(OH)₂）：CaCO₃ + 2CH₃COOH = (CH₃COO)₂Ca + H₂O + CO₂↑
▶ 酸的通性（H⁺性质，中考必考详细版）
1. 指示剂变色（变色的是指示剂！）：
• 紫色石蕊试液：遇酸变红色
• 无色酚酞试液：遇酸不变色（无色）
• 注意：是酸使指示剂变色，而不是指示剂使酸变色；若题目说"酸使酚酞变红"则是错误的
2. 与活泼金属反应（置换反应）：酸 + 金属 → 盐 + 氢气
Zn + 2HCl = ZnCl₂ + H₂↑
• 现象：锌粒逐渐溶解，表面产生大量气泡，放出热量
Fe + H₂SO₄ = FeSO₄ + H₂↑
• 现象：铁钉逐渐溶解，表面产生气泡，溶液由无色变为浅绿色（Fe²⁺），触摸试管感到温热
• 金属活动性要求：必须是H前金属（K、Ca、Na、Mg、Al、Zn、Fe、Sn、Pb），且K、Ca、Na太活泼，与酸反应剧烈，可能发生爆炸，一般不用；H后金属（Cu、Hg、Ag、Pt、Au）不与稀盐酸/稀硫酸反应
• 注意：铁与酸反应生成+2价亚铁盐（浅绿色），不是+3价铁盐（黄色）
• 特例：硝酸（无论浓稀）、浓硫酸与金属反应不生成H₂（氧化性酸），而是生成水、氮氧化物或硫氧化物
3. 与金属氧化物反应（除锈原理）：酸 + 金属氧化物 → 盐 + 水
Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O
• 现象：红棕色铁锈逐渐减少（消失），溶液由无色变为黄色（Fe³⁺）
CuO + H₂SO₄ = CuSO₄ + H₂O
• 现象：黑色固体逐渐减少（消失），溶液由无色变为蓝色（Cu²⁺）
4. 与碱反应（中和反应）：酸 + 碱 → 盐 + 水（放热反应）
HCl + NaOH = NaCl + H₂O
• 现象：若无明显现象（如NaOH与HCl），需借助指示剂判断反应发生；若生成沉淀（如H₂SO₄ + Ba(OH)₂ = BaSO₄↓ + 2H₂O），则观察到白色沉淀
• 应用：①治疗胃酸过多（Mg(OH)₂、Al(OH)₃、NaHCO₃）；②改良酸性土壤（熟石灰Ca(OH)₂）；③处理工厂酸性废水；④蚊虫叮咬涂肥皂水（中和蚁酸）
5. 与某些盐反应（复分解反应）：强酸制弱酸，或生成沉淀/气体/水
CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑（实验室制CO₂）
• 现象：白色固体溶解，产生大量气泡（CO₂能使澄清石灰水变浑浊）
AgNO₃ + HCl = AgCl↓ + HNO₃（检验Cl⁻）
• 现象：产生白色沉淀，该沉淀不溶于稀硝酸
BaCl₂ + H₂SO₄ = BaSO₄↓ + 2HCl（检验SO₄²⁻）
• 现象：产生白色沉淀，该沉淀不溶于稀硝酸
▶ 碱 Bases 电离出的阴离子有且仅有OH⁻ 中考重点
▶ 碱的定义与电离（详细）
碱的定义：电离时产生的阴离子有且仅有氢氧根离子（OH⁻）的化合物
• 关键词"有且仅有"：如Cu₂(OH)₂CO₃（碱式碳酸铜）电离时产生OH⁻，但还产生CO₃²⁻等其他阴离子，因此不是碱，而是碱式盐；又如Mg(OH)Cl是碱式盐而非碱
• 电离方程式示例：
NaOH = Na⁺ + OH⁻
Ca(OH)₂ = Ca²⁺ + 2OH⁻
KOH = K⁺ + OH⁻
NH₃·H₂O ⇌ NH₄⁺ + OH⁻（可逆，部分电离，弱碱）
• 注意：碱中一定含有氢、氧两种元素（OH⁻），但含有氢氧元素的不一定是碱（如H₂O、碱式盐）
▶ 初中四大碱（必须掌握详细性质）
▶ 氢氧化钠 NaOH（烧碱/火碱/苛性钠） 高频
• 物理性质：白色片状或块状固体，易溶于水，溶解时放出大量的热（溶液温度显著升高），有强吸水性（潮解性），暴露在空气中易吸收水分而逐渐溶解，有强腐蚀性（若不慎沾到皮肤，先用大量水冲洗，再涂硼酸溶液）
• 化学性质1：与CO₂反应（变质原理，中考必考）：
2NaOH + CO₂ = Na₂CO₃ + H₂O
• 现象：无明显现象（因此需用其他方法检验变质，如加酸看气泡），因此NaOH必须密封保存，防止变质
• 化学性质2：与SO₂反应（新增，工业脱硫）：
2NaOH + SO₂ = Na₂SO₃ + H₂O
• 应用：吸收工业废气中的SO₂，防止酸雨
• 若SO₂过量：Na₂SO₃ + SO₂ + H₂O = 2NaHSO₃
• 用途：制肥皂、造纸、纺织、印染、精炼石油、用作干燥剂（不能干燥酸性气体如CO₂、HCl、SO₂）
▶ 氢氧化钙 Ca(OH)₂（熟石灰/消石灰） 高频
• 物理性质：白色粉末状固体，微溶于水（溶解度随温度升高而降低，特例！），水溶液俗称石灰水，悬浊液俗称石灰乳/石灰浆；对皮肤有腐蚀性
• 制备：生石灰与水反应：CaO + H₂O = Ca(OH)₂（放热反应，放出大量的热）
• 化学性质1：与CO₂反应（检验CO₂，中考必考）：
Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O
• 现象：澄清石灰水变浑浊（产生白色沉淀）
• 继续通CO₂：CaCO₃ + H₂O + CO₂ = Ca(HCO₃)₂，沉淀溶解，溶液变澄清（碳酸氢钙可溶）
• 化学性质2：与酸反应（改良酸性土壤）：Ca(OH)₂ + 2HCl = CaCl₂ + 2H₂O
• 化学性质3：与Na₂CO₃反应（工业制烧碱）：
Ca(OH)₂ + Na₂CO₃ = CaCO₃↓ + 2NaOH
• 用途：①检验CO₂；②改良酸性土壤（中和土壤酸性）；③砌砖抹墙（与空气中CO₂反应变硬：Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O）；④配制波尔多液（与CuSO₄混合）；⑤处理硫酸厂废水
▶ 氨水 NH₃·H₂O
• 物理性质：氨气的水溶液，无色液体，有强烈的刺激性气味（氨臭味），易挥发（需密封保存），密度比水小
• 化学性质：弱碱性，使酚酞变红；与酸反应：NH₃·H₂O + HCl = NH₄Cl + H₂O
• 特殊用途：制备Al(OH)₃时只能用氨水（弱碱），不能用NaOH（强碱），因为Al(OH)₃溶于强碱：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
▶ 难溶性碱（掌握颜色和受热分解）
• 氢氧化铜 Cu(OH)₂：蓝色絮状沉淀，不溶于水，受热分解：Cu(OH)₂ = CuO（黑色） + H₂O（加热）
• 氢氧化铁 Fe(OH)₃：红褐色沉淀，不溶于水，受热分解：2Fe(OH)₃ = Fe₂O₃（红棕色） + 3H₂O（加热）
• 氢氧化亚铁 Fe(OH)₂：白色絮状沉淀，不溶于水，在空气中极易被氧化：白色 → 灰绿色 → 红褐色（最终变为Fe(OH)₃）
• 氢氧化镁 Mg(OH)₂：白色沉淀，不溶于水，用于治疗胃酸过多（抗酸药）：Mg(OH)₂ + 2HCl = MgCl₂ + 2H₂O
▶ 碱的通性（OH⁻性质，中考必考详细版）
1. 指示剂变色：
• 紫色石蕊试液：遇碱变蓝色
• 无色酚酞试液：遇碱变红色（遇浓碱先变红后褪色，因强氧化性）
2. 与非金属氧化物反应（生成盐和水）：
2NaOH + CO₂ = Na₂CO₃ + H₂O
• 应用：解释NaOH必须密封保存的原因（吸收CO₂变质）
Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O
• 应用：检验CO₂，澄清石灰水变浑浊
2NaOH + SO₂ = Na₂SO₃ + H₂O
• 应用：吸收工业废气中的SO₂，防止酸雨
3. 与酸反应（中和反应）：
HCl + NaOH = NaCl + H₂O
• 现象：若无明显现象需用指示剂；若难溶性碱与酸反应则固体溶解
• 应用：熟石灰改良酸性土壤；氢氧化铝治疗胃酸过多：Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O
4. 与某些盐反应（复分解反应，反应物均需可溶）：
2NaOH + CuSO₄ = Cu(OH)₂↓（蓝色） + Na₂SO₄
• 现象：产生蓝色絮状沉淀
3NaOH + FeCl₃ = Fe(OH)₃↓（红褐色） + 3NaCl
• 现象：产生红褐色沉淀
Ca(OH)₂ + Na₂CO₃ = CaCO₃↓（白色） + 2NaOH
• 应用：工业制烧碱（纯碱制烧碱）
▶ 氢氧化铝 Al(OH)₃（两性氢氧化物） 初中了解 高中系统
初中要求：
• 物理性质：白色胶状沉淀，难溶于水
• 制备：只能用氨水（弱碱）与铝盐反应制备，不能用NaOH等强碱，因为Al(OH)₃能溶于强碱
• 用途：①治疗胃酸过多（弱碱性，不损伤胃壁）：Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O；②净水剂（明矾[KAl(SO₄)₂·12H₂O]溶于水生成Al(OH)₃胶体吸附悬浮杂质）
高中拓展（两性）：
• 与酸反应：Al(OH)₃ + 3HCl = AlCl₃ + 3H₂O
• 与碱反应：Al(OH)₃ + NaOH = NaAlO₂ + 2H₂O（偏铝酸钠）或写作Na[Al(OH)₄]（四羟基合铝酸钠）
• 热分解：2Al(OH)₃ = Al₂O₃ + 3H₂O（加热）

▶ 初中常见盐（详细掌握）
▶ 氯化钠 NaCl（食盐）
• 物理性质：白色晶体，易溶于水，水溶液呈中性（pH=7），熔点801°C，沸点1413°C
• 来源：海水晒盐（蒸发结晶）、盐湖、盐矿（井盐、岩盐）
• 粗盐提纯步骤：①溶解（加水搅拌）；②过滤（除不溶性杂质）；③蒸发（结晶）；④计算产率
• 用途：①调味（0.9%生理盐水）；②农业选种（用16%NaCl溶液，饱满种子下沉）；③化工原料（制Na₂CO₃、Cl₂、NaOH等）；④腌渍食品（防腐，使细菌脱水死亡）
▶ 碳酸钠 Na₂CO₃（纯碱/苏打） 高频
• 物理性质：白色粉末，易溶于水，水溶液呈碱性（pH>7，使酚酞变红），有涩味
• 俗名由来：因其水溶液呈碱性，且碱性较强，故称"纯碱"，但实质是盐（由金属离子和酸根离子构成），不是碱
• 化学性质：
① 与酸反应：Na₂CO₃ + 2HCl = 2NaCl + H₂O + CO₂↑（产生气泡）
② 与碱反应：Na₂CO₃ + Ca(OH)₂ = CaCO₃↓ + 2NaOH（工业制烧碱）
③ 与盐反应：Na₂CO₃ + CaCl₂ = CaCO₃↓ + 2NaCl（产生白色沉淀）
• 用途：玻璃制造（与SiO₂反应）、造纸、纺织、洗涤剂（去油污）、食品加工（中和面团酸味）
▶ 碳酸氢钠 NaHCO₃（小苏打） 高频
• 物理性质：白色晶体，可溶于水，水溶液呈弱碱性（比Na₂CO₃碱性弱）
• 化学性质1：受热分解（重点）：
2NaHCO₃ = Na₂CO₃ + H₂O + CO₂↑（加热）
• 现象：产生能使澄清石灰水变浑浊的气体，试管口有水珠生成
• 化学性质2：与酸反应：NaHCO₃ + HCl = NaCl + H₂O + CO₂↑（反应比Na₂CO₃剧烈，产生气泡速度快）
• 用途：①发酵粉（与酸反应或受热产生CO₂，使糕点疏松多孔）；②治疗胃酸过多（碱性弱，对胃刺激小）；③灭火器原料（与Al₂(SO₄)₃反应产生CO₂）；④制汽水
▶ 碳酸钙 CaCO₃（石灰石/大理石） 高频
• 物理性质：白色固体，难溶于水，是大理石、石灰石、方解石、白垩的主要成分
• 化学性质1：与酸反应（实验室制CO₂）：
CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
• 现象：白色固体逐渐溶解，产生大量气泡，生成能使澄清石灰水变浑浊的气体
• 化学性质2：高温分解（工业制生石灰）：
CaCO₃ = CaO + CO₂↑（高温）
• 用途：①建筑材料（天安门华表、大理石地板）；②补钙剂（咀嚼片）；③实验室制CO₂；④工业制生石灰和CO₂
• 溶洞形成：CaCO₃ + H₂O + CO₂ ⇌ Ca(HCO₃)₂（可逆反应，碳酸氢钙可溶于水），钟乳石、石笋的形成是此反应的逆过程
▶ 硫酸铜 CuSO₄
• 物理性质：无水CuSO₄为白色粉末，易溶于水；CuSO₄·5H₂O（五水合硫酸铜，胆矾/蓝矾）为蓝色晶体
• 特性：无水CuSO₄遇水变蓝（生成CuSO₄·5H₂O），用于检验H₂O的存在
• 化学性质：
① 与碱：CuSO₄ + 2NaOH = Cu(OH)₂↓（蓝色） + Na₂SO₄
② 与金属（置换）：Fe + CuSO₄ = FeSO₄ + Cu（湿法炼铜，铁表面变红，溶液变浅绿）
• 用途：①波尔多液[Ca(OH)₂ + CuSO₄]杀菌剂（农业防病虫害，不能用铁桶盛装）；②精炼铜；③游泳池消毒
▶ 盐的化学性质（复分解反应条件，中考必考详细版）
1. 与金属反应（置换反应）：前置后，盐需溶（K、Ca、Na除外，因太活泼先与水反应）
Fe + CuSO₄ = FeSO₄ + Cu
• 现象：铁钉表面逐渐覆盖一层红色固体（铜），溶液由蓝色逐渐变为浅绿色（Fe²⁺）
• 应用：湿法炼铜；不能用铁桶装波尔多液或硫酸铜溶液
Cu + 2AgNO₃ = Cu(NO₃)₂ + 2Ag
• 现象：铜丝表面逐渐覆盖一层银白色固体（银），溶液由无色逐渐变为蓝色（Cu²⁺）
• 注意：铁与盐溶液反应生成+2价亚铁盐（浅绿色）
2. 与酸反应（强酸制弱酸，或生成沉淀/气体/水）：
CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
• 现象：白色固体溶解，产生大量气泡（CO₂能使澄清石灰水变浑浊），用于检验碳酸盐
AgNO₃ + HCl = AgCl↓ + HNO₃（检验Cl⁻）
• 现象：产生白色沉淀，该沉淀不溶于稀硝酸
BaCl₂ + H₂SO₄ = BaSO₄↓ + 2HCl（检验SO₄²⁻）
• 现象：产生白色沉淀，该沉淀不溶于稀硝酸
3. 与碱反应（复分解，反应物均需溶，生成沉淀）：
CuSO₄ + 2NaOH = Cu(OH)₂↓（蓝色） + Na₂SO₄
• 现象：产生蓝色絮状沉淀
Ca(OH)₂ + Na₂CO₃ = CaCO₃↓（白色） + 2NaOH
• 应用：工业制烧碱（纯碱制烧碱）
4. 与盐反应（复分解，反应物均需溶，生成沉淀）：
NaCl + AgNO₃ = AgCl↓ + NaNO₃
Na₂SO₄ + BaCl₂ = BaSO₄↓ + 2NaCl
Na₂CO₃ + CaCl₂ = CaCO₃↓ + 2NaCl
▶ 盐的溶解性规律（口诀必背）
溶解性口诀（中考必背）：
钾钠铵硝皆可溶（K⁺、Na⁺、NH₄⁺、NO₃⁻的盐都溶于水）
盐酸盐不溶银亚汞（AgCl、Hg₂Cl₂不溶于水，其余盐酸盐都溶）
硫酸盐不溶钡和铅（BaSO₄、PbSO₄不溶于水，CaSO₄、Ag₂SO₄微溶，其余硫酸盐都溶）
碳酸盐只溶钾钠铵（K₂CO₃、Na₂CO₃、(NH₄)₂CO₃可溶，其余碳酸盐难溶，但可溶于酸）
碱溶钾钠钡钙铵（KOH、NaOH、Ba(OH)₂可溶，Ca(OH)₂微溶，NH₃·H₂O可溶，其余碱难溶）
酸多可溶（常见酸一般可溶）
特殊记忆（中考高频）：
• AgCl和BaSO₄：白色沉淀，不溶于稀硝酸，用于分别检验Cl⁻和SO₄²⁻
• CaCO₃、BaCO₃：白色沉淀，溶于稀硝酸且产生气泡（CO₂）
• Fe(OH)₃：红褐色沉淀；Cu(OH)₂：蓝色沉淀；Mg(OH)₂：白色沉淀
▶ 金属 Metals 单质 中考重点
▶ 金属活动性顺序（必背，浙江中考必考） 核心
K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
金属活动性由强逐渐减弱（还原性减弱）
应用规律（中考必考）：
1. 位置越靠前，金属活动性越强，越易失去电子，化学性质越活泼
2. 位于H前面的金属能与稀盐酸/稀硫酸反应置换出氢气（K、Ca、Na除外，因太活泼，与水反应剧烈）
3. 位于前面的金属能把位于后面的金属从其盐溶液中置换出来（K、Ca、Na除外，先与水反应）
4. 铁参与置换反应生成+2价亚铁盐（浅绿色），不是+3价铁盐（黄色）
▶ 初中重点金属单质（详细）
▶ 铁 Fe（中考重中之重）
物理性质：银白色金属光泽（铁粉为黑色），质软（纯铁），有良好的延展性、导电性和导热性，能被磁化（吸引铁、钴、镍等物质），熔点1535°C，沸点2750°C
化学性质1：与氧气反应（条件不同产物不同）：
3Fe + 2O₂ = Fe₃O₄（点燃，火星四射）
• 现象：剧烈燃烧，火星四射，放出大量的热，生成黑色固体（Fe₃O₄，四氧化三铁）
• 注意：铁在氧气中燃烧生成Fe₃O₄（磁性氧化铁），不是Fe₂O₃！Fe₃O₄可看作FeO·Fe₂O₃，含+2价和+3价铁
4Fe + 3O₂ = 2Fe₂O₃（缓慢氧化，铁生锈）
化学性质2：与酸反应：
Fe + 2HCl = FeCl₂ + H₂↑
• 现象：铁钉逐渐溶解，表面产生气泡，溶液由无色变为浅绿色（Fe²⁺）
• 注意：生成的是+2价亚铁盐（浅绿色），不是+3价铁盐（黄色）
化学性质3：与盐溶液反应（湿法炼铜）：
Fe + CuSO₄ = FeSO₄ + Cu
• 现象：铁钉表面逐渐覆盖一层红色固体（铜），溶液由蓝色逐渐变为浅绿色
• 应用：古代湿法炼铜；不能用铁桶装波尔多液或硫酸铜溶液（会腐蚀铁桶并降低药效）
铁生锈（中考实验探究高频）：
• 实质：铁与O₂、H₂O同时接触发生的复杂缓慢氧化反应，产物为Fe₂O₃·nH₂O（铁锈，疏松多孔，红棕色）
• 防锈原理：隔绝O₂和H₂O中的至少一种
• 方法：涂油（机械零件）、刷漆（门窗、栏杆）、镀金属（镀锌、镀铬）、烤蓝（锯条）、制成不锈钢（加Cr、Ni）
冶炼（高炉炼铁，还原法）：
3CO + Fe₂O₃ = 2Fe + 3CO₂（高温）
• 原料：铁矿石（Fe₂O₃）、焦炭（燃烧提供热量并制CO）、石灰石（除SiO₂造渣）、热空气
• 产物：生铁（含碳2%-4.3%，不是纯铁），不是钢
▶ 铝 Al（重点：氧化膜保护）
物理性质：银白色轻金属（密度2.7g/cm³，小于铁），具有良好的延展性、导电性（仅次于银、铜）和导热性，熔点660°C
核心性质：抗腐蚀性（中考必考，详细机制）
• 原因：铝在空气中与氧气反应，表面生成一层致密的氧化铝（Al₂O₃）氧化膜，这层氧化膜非常致密，能牢固地覆盖在铝表面，阻止内部的铝继续与氧气反应，从而保护铝不再被氧化
4Al + 3O₂ = 2Al₂O₃（常温下自然形成）
• 特性：这层氧化膜不仅能阻止铝与氧气反应，还能阻止铝与水反应，因此铝制品在空气中非常稳定，具有良好的抗腐蚀性
• 破坏与再生：若氧化膜被破坏（如用砂纸打磨），铝会立即与氧气反应重新生成氧化膜
• 应用：铝合金门窗、易拉罐、飞机材料、高压锅、铝箔（香烟包装）
化学性质：
• 与酸：2Al + 6HCl = 2AlCl₃ + 3H₂↑（氧化膜被酸溶解后，内部的铝与酸反应）
• 与碱（高中）：2Al + 2NaOH + 2H₂O = 2NaAlO₂ + 3H₂↑，说明铝具有一定的两性
• 铝热反应（高中）：2Al + Fe₂O₃ = 2Fe + Al₂O₃（高温，放出大量的热，用于焊接铁轨）
▶ 铜 Cu
物理性质：紫红色（红色）金属，有良好的延展性，导电性仅次于银（常用作导线），导热性好，熔点1083°C
化学性质：
• 与氧气：2Cu + O₂ = 2CuO（加热，铜丝表面变黑）
• 生锈（铜绿）：2Cu + O₂ + H₂O + CO₂ = Cu₂(OH)₂CO₃（碱式碳酸铜，绿色，铜器表面铜绿）
• 与强氧化性酸（高中）：Cu与浓硫酸、浓硝酸反应，但不与稀盐酸、稀硫酸反应（H后金属）
▶ 钠 Na、钾 K（初中了解，高中重点） 高中重点
• 物理性质：银白色金属，质软（可用刀切割），密度小于水（浮在水面），熔点低（Na为97.8°C），保存在煤油中（隔绝空气和水）
• 化学性质：极活泼，与水剧烈反应
2Na + 2H₂O = 2NaOH + H₂↑
• 现象：浮（密度小于水）、熔（反应放热，熔点低）、游（产生气体推动）、响（反应剧烈）、红（加酚酞变红，生成NaOH）
• 与O₂：常温生成Na₂O（白色），加热生成Na₂O₂（淡黄色，过氧化钠，高中重点）
▶ 金属材料与合金（详细）
• 合金定义：金属与金属（或非金属）熔合而成的具有金属特性的物质，属于混合物
• 合金特性（重点）：①熔点一般低于成分金属（如武德合金熔点低作保险丝）；②硬度一般大于成分金属（如硬铝比纯铝硬）；③抗腐蚀性强于成分金属（如不锈钢比纯铁耐腐蚀）
• 生铁 vs 钢：都是铁碳合金，区别在含碳量：生铁2%-4.3%（硬而脆，可铸不可锻），钢0.03%-2%（硬而韧，可铸可锻）
• 重要合金：①不锈钢（加Cr、Ni，抗腐蚀）；②武德合金（低熔点，保险丝）；③黄铜（Cu-Zn）；④青铜（Cu-Sn）；⑤硬铝（Al-Cu-Mg-Mn-Si，航空材料）

钙系列转化链（必会详细版）：
① CaCO₃ = CaO + CO₂↑（高温煅烧，工业制生石灰和CO₂）
• 现象：白色固体分解，生成能使澄清石灰水变浑浊的气体
② CaO + H₂O = Ca(OH)₂（放热反应，生石灰变熟石灰）
• 现象：块状固体变为粉末，放出大量的热，温度升高
③ Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O（检验CO₂，石灰水变浑浊）
• 现象：澄清石灰水变浑浊，产生白色沉淀
④ Ca(OH)₂ + Na₂CO₃ = CaCO₃↓ + 2NaOH（工业制烧碱）
• 现象：产生白色沉淀
⑤ CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑（实验室制CO₂，检验碳酸盐）
• 现象：白色固体溶解，产生气泡
铜系列转化链（推断题常考）：
① 2Cu + O₂ = 2CuO（加热，紫红色固体表面变黑）
② CuO + H₂SO₄ = CuSO₄ + H₂O（黑色固体逐渐减少，溶液变蓝）
③ CuSO₄ + 2NaOH = Cu(OH)₂↓ + Na₂SO₄（产生蓝色沉淀）
④ Cu(OH)₂ = CuO + H₂O（加热，蓝色沉淀变黑）
⑤ CuO + H₂ = Cu + H₂O（加热，黑色固体变红，试管口有水珠）
⑥ Fe + CuSO₄ = FeSO₄ + Cu（铁表面变红，溶液变浅绿）
碳系列转化链：
① 2C + O₂(不足) = 2CO（不完全燃烧）
② 2CO + O₂ = 2CO₂（点燃，蓝色火焰）
③ CO₂ + H₂O = H₂CO₃（使石蕊变红）
④ H₂CO₃ = H₂O + CO₂↑（分解，红色褪去）
⑤ CO₂ + Ca(OH)₂ = CaCO₃↓ + H₂O（石灰水变浑浊）
⑥ CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
推断题突破口（特征颜色/现象必背）：
• 颜色：红色（Cu、Fe₂O₃）、黑色（CuO、Fe₃O₄、C、MnO₂、铁粉）、蓝色（Cu(OH)₂、CuSO₄溶液）、浅绿色（Fe²⁺）、黄色（Fe³⁺）、紫红色（KMnO₄）
• 沉淀：蓝色Cu(OH)₂、红褐色Fe(OH)₃、白色Mg(OH)₂/BaSO₄/AgCl/CaCO₃
• 气体：唯一碱性气体NH₃（使湿润红色石蕊试纸变蓝）
• 特征反应：不溶于稀硝酸的白色沉淀只有AgCl和BaSO₄；溶于酸冒泡的是碳酸盐
